HTML basics

  • !DOCTYPE html must be declared at the top of every HTML code.
  • The html tags must surround the whole HTML code
  • An opening tag is surrounded by <...>
  • A closing tag is surrounded by </...>
  • Caution! Not all tags have opening and closing tags

Setting up a document

HTML Tag Description
<head> Surrounds the header of an HTML document. Contains metadata and link to CSS code
<title> Goes inside the head to display the title of the page in the browser.
<div> Divides HTML code into sections to provide easier access using classes and ids
<main> Encapsulates dominant content
<h1> Tag for headed. h1 through h6 can be used. h1 being the largest, h6 the smallet header type
<br> Break: inserts a new line.
<p> Surrounds a paragraph / block of text
<footer> Contains page footer
