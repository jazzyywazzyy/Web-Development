Basics of CSS

Semantics of CSS

Name Description
Selector p{color: blue;} declares which element will be styled
Declaration block p{color: blue;} contains declarations
Declaration p{color: blue;} value pair that applies style to an element
Property p{color: blue;} signifies what visual characteristic of element is to be modified
Value p{color:blue;} signifies value of property

Linking CSS document to an HTML document:

  • <link> within the HTML document head
  • href = "address / path of CSS document"
  • rel = "stylesheet" (description of the relation)
  • type= "text/css" (type of document)
Link Description Example
.attribute{} accesses class attributes <p class = "name">
#attribute accesses id attributes <h1 id="Introduction">

Order of specification (highest to lowest): IDs, class, type
